# Termhabit

- termhabit is a habit and todo manager in your terminal

# TermHabit Customization

**TermHabit** allows you to track habits directly in your terminal.
While the default appearance is simple, you can customize various aspects to better suit your workflow.

## Colors

You can either use **LipGloss color codes** or standard **HEX color codes**

- **Title**
- **Cursor**
- **Input**
- **Border**
